* Additional Data Protection Clauses:

In addition to the permissions and limitations granted by the BSD 3-clause
License, operators of the Telegram bot, whether self-hosting or deploying it on
a server, are subject to the following data protection restrictions:

1. User Message Storage: Operators are prohibited from storing any of the
following, whether plain text or encrypted:
- User-generated messages to the bot
- User-received messages from the bot

2. User Information Storage: Operators are prohibited from storing any of the
following user information, whether plain text or encrypted:
- User's first name
- User's last name
- User's username
- User's profile photos
- Any other personal information provided by the Telegram API

server may store in an encrypted or hashed format if strictly necessary for
operational purposes.
